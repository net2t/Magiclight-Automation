OAuth 2.0 — Step by Step
1
Enable YouTube Data API v3
Google Cloud Console → APIs & Services → Library → Enable "YouTube Data API v3"
↗ Open Google Console
2
Create OAuth 2.0 Credentials
Console → Credentials → Create → OAuth Client ID → Web Application. Add http://localhost as redirect URI.
3
Authorize Your Account
Click below — a Google sign-in window will open. Sign in with the YouTube channel owner's Google account.
4
Paste Access Token
After sign-in, copy the access_token= value from the redirect URL bar and paste it here.
5
Verify & Test
Test the token by fetching your channel info from YouTube API.
