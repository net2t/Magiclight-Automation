SETUP GUIDE
MAGICLIGHT PRO V3 — FULL DOCUMENTATION
✅ HOW 2-WAY WORKS:
READ (CSV) → Dashboard directly fetches your Google Sheet CSV — no login, no Apps Script needed. Instant.
WRITE (Apps Script) → Edit/Delete/Upload/Schedule all go through Apps Script backend. One-time deploy.
01
GOOGLE CLOUD SETUP
1. Go to console.cloud.google.com
2. Create new project: MagicLight-Pro
3. APIs & Services → Library
4. Search & Enable: YouTube Data API v3
5. OAuth consent screen → Add your Gmail as test user
6. Do this separately for each YouTube account
02
APPS SCRIPT DEPLOY
1. Open your Google Sheet
2. Extensions → Apps Script
3. Replace Code.gs with the provided file
4. Set SHEET_ID at top of Code.gs
5. Services (+) → Add YouTube Data API v3
6. Deploy → New Deployment → Web App
   Execute as: Me · Access: Anyone
7. Copy URL → paste in Config Tab
03
FIRST AUTHORIZATION
1. Apps Script editor → function dropdown
2. Select authorizeYouTube → Run
3. Google popup → Allow all permissions
4. Select setupTrigger → Run
   (This creates hourly auto-upload trigger)
5. Test: run doGet → should return JSON OK
04
GOOGLE SHEET CSV URL
1. Open your Google Sheet
2. File → Share → Publish to web
3. Select your sheet tab → CSV format
4. Click Publish → copy the URL
5. Paste in Config Tab → CSV URL
6. This URL is used for READ (no auth needed)
05
ADD EXTRA ACCOUNTS
1. Accounts Tab → + ADD ACCOUNT
2. Enter Gmail + channel name
3. An Auth URL will appear
4. Open that URL in browser while logged into
   the CORRECT Google/YouTube account
5. Allow permissions → token saved
6. Green dot = authorized ✓
06
WRONG ACCOUNT POSTING?
Problem: Videos posting from wrong Gmail account

Fix:
1. Accounts Tab → find the wrong account
2. Click DE-AUTH → token cleared
3. An Auth URL opens automatically
4. Open in browser, switch to correct Gmail
5. Allow → correct token saved
6. Now uploads go to correct channel ✓

QUOTA LIMITS
YouTube API daily quota: 10,000 units

Per video upload: ~1,600 units
Per thumbnail set: ~50 units
Per analytics read: ~1 unit

Max ~5 videos/day per Cloud Project

💡 Each Gmail account can have its own
Google Cloud Project → own 10,000 quota!
COMMON ERRORS
"caller does not have permission"
→ OAuth scope missing. Re-run authorizeYouTube() in Apps Script
"Daily Limit Exceeded"
→ Quota finished. Wait 24hrs or create new Cloud Project
"File not found" on Drive
→ Set Drive file sharing to "Anyone with link — Viewer"
Thumbnail not uploading
→ Channel needs 1000+ subscribers for custom thumbnails
CSV not loading
→ Sheet must be published to web as CSV. Check Config tab URL
Video not showing on YouTube
→ Processing takes 5-15 min after upload. Normal.
