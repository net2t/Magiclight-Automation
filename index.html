Tips & YouTube Guide

📋 Title Best Practices

Keep titles under 60 characters. Front-load your keyword. Avoid ALL CAPS. Example: "Raya and the Kite — A Story About Friendship" is better than "AMAZING KITE STORY FOR KIDS!!!"

🏷️ Tags Strategy

Use 10-15 tags. Mix broad (kids story, animated story) and specific (teamwork story, friendship lesson). Add your channel name as a tag. Don't keyword stuff — YouTube's algorithm ignores spam tags.

📸 Thumbnail Rules

Optimal size: 1280×720px (16:9). File under 2MB. JPG or PNG. Your Drive thumbnail link (Col N) will be auto-used. Make sure file is shared as "Anyone with link."

⚠️ API Quota Limits

YouTube Data API v3 gives 10,000 units/day per project. One video upload costs ~1,600 units. That means ~6 uploads/day per Google Cloud project. Use separate projects for each account to avoid quota hits.

📅 Best Upload Schedule (Pakistan)

8-10 AM PKT: Great for kids/family content (school morning). 6-8 PM PKT: Prime time for all ages. Friday and Saturday evenings get 30-40% more views. Avoid 2-5 AM uploads.

🔑 OAuth Setup (Multi-Account)

1. Go to console.cloud.google.com — create ONE project per YouTube account. 2. Enable YouTube Data API v3. 3. Create OAuth 2.0 credentials (Desktop App). 4. In Apps Script → Project Settings → Script Properties, add: OAUTH_TOKEN_Teamwork, OAUTH_TOKEN_Adventure, etc. 5. Run authorizeAccount() function once per account to get tokens.

✅ Pre-Upload Checklist

□ Video is MP4 (H.264 codec) □ Drive file is "Anyone with link — Viewer" □ Thumbnail is 1280×720 □ Title under 100 chars □ Description under 5000 chars □ Tags comma-separated □ Category selected □ Privacy set □ "Made for Kids" toggle set correctly

// Set daily trigger
ScriptApp.newTrigger('autoUpload')
  .timeBased().everyDays(1)
  .atHour(10).create();

// Get Drive file as blob
var file = DriveApp.getFileById(fileId);
var blob = file.getBlob();

// Upload to YouTube
YouTube.Videos.insert(resource,
  'snippet,status', blob);
